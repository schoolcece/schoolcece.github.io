mysql服务关闭与启动

关闭: net stop mysql80

启动：net start mysql80

mysql常用操作

  1. 连接用户名为root的数据库服务：mysql -uroot -p，然后输入密码
  2. show databases;：列出所有数据库
  3. create database name;：创建名为name的数据库
  4. drop database name;：删除名为name的数据库
  5. use name;进入名为name的数据库
  6. show tables;：列出当前数据库的所有表
  7. create table user(id int , username varchar(100));创建名为user的表，表中包含整型id和字符数组username两个属性
  8. drop table user;：删除名为user的表
  9. insert into user values(1, 'a');：在上面创建的user表中插入数据
  10. select * from user;：查询表中所有数据
  11. select * from user where id=1;：查询表中id为1的数据
  12. delete from user where id=1;：删除表中id为1的数据
